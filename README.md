# Aria - Local AI Assistant

## Overview
Aria is a locally running AI assistant built using Python and Ollama.
It combines conversational AI with system-level command execution.

## Features
- Natural conversation with memory
- Open apps and websites (YouTube, Spotify, etc.)
- YouTube search automation
- WhatsApp web fallback
- Folder navigation and system control

## Tech Stack
- Python
- Ollama (LLM)
- Requests
- OS / subprocess automation
